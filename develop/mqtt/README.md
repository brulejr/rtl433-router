# MQTT Docker Setup

This provides a containerized MQTT broker service using Docker.

### Usage
Run using the Docker compose plugin.

```bash
docker compose up -d
```

Or, run using Docker Compose itself.
```bash
docker-compose up -d
```

### Services

---------|------|
mqtt     | 1883 |
